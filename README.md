# FUTURE_CS_02
# Phishing Email Detection & Awareness System

# Objective
To analyze phishing email samples and create awareness to prevent phishing attacks.

## 🛠 Tools Used
- Google Admin Toolbox (Message Header Analyzer)
- MXToolbox
- Browser tools

## Steps Performed
- Collected phishing email samples
- Performed content analysis
- Conducted header analysis
- Identified phishing indicators
- Classified risk levels

## Outcome
This project demonstrates how phishing emails can be detected using both technical (header) and non-technical (content) analysis.
